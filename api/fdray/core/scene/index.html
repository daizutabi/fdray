    Render the scene with the given image dimensions.

    Parameters

    • width : int | None — The width of the image.

    • height : int | None — The height of the image.

    • output_alpha : bool | None — If True, output an image with an alpha channel.

    • quality : int | None — The quality of the image.

    • antialias : bool | float | None — The antialiasing level.

    • threads : int | None — The number of threads to use.

    • trim : bool | int — If True, trim the output image to the non-transparent region. If an integer, trim the output image to the non-transparent region by the given margin.

    Returns

    • Image.Image — The rendered image.
